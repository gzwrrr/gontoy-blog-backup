# Spring 核心注解

[[toc]]


| 注解            | 描述                           |
| --------------- | ------------------------------ |
| @Autowired      | 根据类型自动装配依赖对象       |
| @Component      | 将类交给 Spring 管理           |
| @Configuration  | 标记当前类为配置类             |
| @Controller     | 标记当前类为 Spring MVC 控制器 |
| @RequestMapping | 处理 HTTP 请求                 |
| @Repository     | 标记当前类为数据仓库           |
| @Service        | 标记当前类为业务逻辑组件       |
| @Value          | 将属性注入值                   |
| @Aspect         | 声明一个切面                   |
| @Transactional  | 标记当前方法或类是事务性的     |
