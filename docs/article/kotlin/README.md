# Kotlin

[[toc]]



:::info 说明

Kotlin 学习笔记，持续增加...

Kotlin 是一种在 Java 虚拟机上运行的静态类型编程语言，被称之为 Android 世界的Swift，由 JetBrains 设计开发并开源。

最好有 Java 基础（[戳这里了解更多](../java/)）

:::



## 知识点

- 基本数据类型
- 流程控制：条件分支、循环控制
- 面向对象：继承、封装、多态
- 接口、扩展函数、内置函数、中缀函数
- 数据类、密封类、枚举类、伴生对象
- 泛型
- 委托
- lambda 表达式、内联
- 顶级成员、默认参数、解构声明
- Sequences 序列
- Void、Unit、Nothing
- DSL 领域特定语言
