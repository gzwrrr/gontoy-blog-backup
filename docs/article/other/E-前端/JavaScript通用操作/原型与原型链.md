# 原型链


原型：prototype

原型链（链接点）：_\_proto__



从属关系：

prototype 是函数的一个属性，本质是一个对象

_\_proto__ 是对象 Object 的一个属性，本质也是一个对象



**对象的 _\_proto__ 保存着该对象的构造函数的 prototype**
