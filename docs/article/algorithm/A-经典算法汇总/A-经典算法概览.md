## 常用的算法

> 基本上带名字的算法都不考，但是原理要知道

常用：

1. 二分法
2. 拓扑排序
3. 动态规划
4. 分治法
5. 贪心
6. 最小生成树

不常用：

1. 最短路算法（Dijkstra、Floyd）
2. Morris 遍历
3. Manacher
4. KMP
5. 网络流
6. 希尔排序





## 常用的数据结构

常用：

1. 哈希表
2. 二叉搜索树
3. 堆
4. 并查集
5. 字典树

不常用：

1. 跳表
2. 线段树
3. 平衡排序二叉树（比如红黑树）
4. B/B+ 树
5. KD 树
